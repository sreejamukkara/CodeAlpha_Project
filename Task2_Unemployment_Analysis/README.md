# Unemployment Analysis with Python

## Description
This project analyzes unemployment rate data to understand trends, patterns,
and the impact of Covid-19 using data visualization and analysis techniques.

## Work Done
- Data cleaning and preprocessing
- Exploratory data analysis (EDA)
- Time-based trend analysis
- Comparative analysis before and after Covid-19

## Visualizations
- Line plots showing unemployment trends over time
- Comparison plots highlighting Covid-19 impact
- State-wise or category-wise unemployment analysis

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Outcome
Identified important unemployment trends and visual insights that help
understand economic changes over time.
